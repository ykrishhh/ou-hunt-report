# Security Policy

## Scope

This repository contains the results of an **authorized** red team assessment
of University of Oklahoma (`ou.edu`) infrastructure, conducted for security
research and defensive improvement.

## Responsible Disclosure

If you are affiliated with the target organization and wish to discuss
remediation, open a private security advisory or contact the maintainer.

## Legal

All testing was performed within an authorized scope. Do not use the included
proof-of-concept scripts against systems you do not own or have explicit
written permission to test. The maintainer is not responsible for misuse.

## Reporting a Vulnerability in This Repo

Open an issue or security advisory. Findings are reviewed on a best-effort basis.
